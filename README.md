
# e-plantShopping

## Proyecto: e-plantShopping

e-plantShopping es una aplicación web desarrollada con React que simula una tienda en línea de plantas.  
La aplicación permite visualizar productos, explorar el catálogo y sentar las bases para funcionalidades
como carrito de compras y gestión de productos.

Este repositorio contiene la estructura base del proyecto y será utilizado para el desarrollo
y seguimiento del trabajo durante el curso **Desarrollo de aplicaciones front-end con React**.
